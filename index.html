Project Information

The plan for the final project is to collect, analyze, and process data about European countries. Materials would include the European census, map, geodata, etc. The goal is to analyze population number, number of residents for different population groups (socio-economic, ethnicity, etc.), the income, and other geographical data for European countries for the sake of continental comparison, primarily using Python, and make a visual projection. A choropleth map, graph, and table will and/or may be produced for the sake of this project. To narrow the scope somewhat, the focus will be put on western European countries and/or those which are part of the European Union (EU) for analysis. However, should time allow it, comparison over time (i.e. compare the differences in trends between 2000 and 2010) could also be made, as well as comparison inside the different countries of choice (depending on the codig method, this might be possible).

Minimum of deliverables for the project include:

  • An essay of the research findings;
  • The code used for the project;
  • A map and graphics where needed;
  • A link to the project.
